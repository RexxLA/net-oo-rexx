This is a BSF4ooRexx850 JDORFX command handler that uses JavaFX instead of
awt/swing to create images.

It is based/rooted in JDOR (Java2D for ooRexx) introduced with BSF4ooRexx850.

To use:

        - copy the jar file in "lib" to the "BSF4ooRexx850\lib" directory or
          to "BSF4ooRexx\lib" in the user home directory, such that Java can
          pick the contained class

        - go into the samples dir and run the scripts, e.g.

                rexx jdorfx_camera.rxj

            or on Apple use rexxj2.sh or rexxjh.sh:

                rexxjh.sh jdorfx_camera.rxj

The synopsis of the implemented commands is documented in samples\jdorfx_commands.txt.
